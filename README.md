# PapaTapClone
WARNING: Running this application may potentially trigger seizures for people with photosensitive epilepsy.  
  
A much less detailed version of papatap (https://patatap.com/). Entering any letters on the keyboard prompts a specific sound and color animation.  
Animation implemented using Paper.js and sounds edited using Howler.js. If you enjoy spamming the keyboard, you will enjoy running the app.  
App can be found in index.html.  
Additional flashing lights animation can be found in lights.html.
